Circuitos Digitais e Introdução a C++/C#

Circuitos Digitais

Nesta aula, vamos explorar os conceitos básicos de circuitos digitais, incluindo:

  • Entrada, processamento e saída
  • Portas lógicas (AND, OR, NOT)
  • Tabelas verdade
  • Circuitos elétricos com chaves, fontes, resistores e LEDs
  • Portas lógicas com ICs (7408, 7432, 7404)
  • Equações e diagramas das portas lógicas

Introdução a C++/C#

Também vamos introduzir a linguagem de programação C++/C# e criar um exemplo prático com Arduino:

  1. Declaração de variáveis e constantes
  2. Configuração (entradas, saídas, lógica e matemática)
  3. Repetições

Código de exemplo

      /*
      -----------------------------

      // Declaração de Variáveis e Constantes
      const int LED = 13;
      
      // Configurações
      void setup(){
        pinMode(LED, OUTPUT);
      }
      // Repetições
      void loop(){
        digitalWrite (LED, HIGH);
        delay(1000); // 1 segundo
        
        digitalWrite (LED, LOW);
        delay(1000); // 1 segundo
      }

Conclusão

Nesta aula, você aprendeu os conceitos básicos de circuitos digitais e como criar um programa simples em C++/C# para controlar um LED com Arduino. Esperamos que você tenha gostado da aula e esteja pronto para explorar mais sobre esses tópicos!
